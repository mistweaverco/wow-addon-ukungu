# Ukungu

Ukungu is [swahili](https://en.wikipedia.org/wiki/Swahili_language) and
means mist/fog/haze.

It is the all in one toolkit for Mistweaver Monks.

## Features

### Warnings in chat when in instance

 - Low on mana (30% mana)
 - Out of mana (10% mana)
 - Near death (10% hp)
 - Dead
 - Has aggro
